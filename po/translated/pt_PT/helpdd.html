Configurações do Sistema SolydXK: Ajuda do Driver do Dispositivo

Descrição

O Driver de Dispositivo é uma ferramenta para ajudar os utilizadores na instalação de drivers de hardware de terceiros. Actualmente, os drivers da Nvidia, AMD/ATI e Broadcom podem ser instaladas e permite instalar o kernel PAE para sistemas de 32 bits multi-core. O Driver de Dispositivo só irá listar os dispositivos que são suportados. Ele não irá listar quaisquer outros dispositivos mencionados acima.

Importante

O Driver de Dispositivo só pode instalar drivers que estão disponíveis nos repositórios. Certifique-se de ter uma conexão à Internet se desejar instalar os drivers.

Comandos

Se você preferir instalar os drivers por terminal, você pode usar os seguintes comandos:

 

# ddm -i amd

Isto irá instalar os drivers para sua placa gráfica AMD / ATI. As placas AMD / ATI da versão HD 5000 e superior são suportados pelos drivers. Versões entre 1000 e 5000 usarão os drivers Radeon. Use "ddm -p amd" para remover os drivers do seu sistema.

# ddm -i nvidia

Isso irá seleccionar os drivers correctos para a sua placa gráfica Nvidia. Ele selecionará os drivers do Bumblebee caso você tenha uma placa híbrida (Nvidia e Intel). Use "ddm -p nvidia" para remover os drivers do seu sistema.

# ddm -i open

Você pode usar este comando se quiser reverter para os drivers do Nouveau aberto. Ele removerá todos os drivers proprietários do seu sistema.

# ddm -i broadcom

Este comando irá instalar o driver correto para o adaptador wireless Broadcom. Use "ddm -p broadcom" para remover os drivers do seu sistema.

# ddm -i pae

Isto irá instalar o kernel PAE em sistemas multi-processadores que executam um sistema operativo de 32 bits. Use "ddm -p pae" para remover o kernel PAE do seu sistema. Observe que você não pode remover o kernel quando actualmente você o tem inicializado.

Registos
