SolydXK System Settings: Device Driver Help

Descripció

The Device Driver is a tool to help users installing third-party hardware drivers. Currently Nvidia, AMD/ATI and Broadcom drivers can be installed and allows to install the PAE kernel for multi-core 32-bit systems. The Device Driver will only list those devices that are supported. It will not list any other devices then mentioned above.

Important

The Device Driver can only install drivers which are available in the repositories. Make sure you have an internet connection if you want to install the drivers.

Ordres

Si preferiu instal·lar els controladors des del terminal, podeu usar aquestes ordres:

 

# ddm -i amd

This will install the fglrx drivers for your AMD/ATI graphical card. AMD/ATI cards from version HD 5000 and up are supported by the fglrx drivers. Versions between 1000 and 5000 will use the Radeon drivers. Use "ddm -p amd" to remove the drivers from your system.

# ddm -i nvidia

Això triarà els controladors correctes per a la vostra tarja gràfica Nvidia. També seleccionarà els controladors Bumblebee en el cas que tingueu una tarja híbrida (amb Nvidia i Intel). Useu "ddm -p nvidia" per eliminar els controladors del vostre sistema.

# ddm -i open

Podeu usar aquesta ordre per a reinstal·lar els controladors lliures Nouveau. Això eliminarà els controladors propietaris del vostre sistema.

# ddm -i broadcom

Aquesta ordre instal·larà el controlador adequat per al vostre adaptador sense fils Broadcom. Useu "ddm -p broadcom" per eliminar aquests controladors del vostre sistema.

# ddm -i pae

Això instal·larà el nucli (kernel) PAE en sistemes multiprocessador amb S. O. de 32 bit. Useu "ddm -p pae" per eliminar el kernel PAE del vostre sistema. Tingueu en compte que no es pot eliminar el nucli que s'està executant en aquell moment.
