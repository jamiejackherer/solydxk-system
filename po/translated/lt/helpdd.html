SolydXK System Settings: Device Driver Help

Aprašas

The Device Driver is a tool to help users installing third-party hardware drivers. Currently Nvidia, AMD/ATI and Broadcom drivers can be installed and allows to install the PAE kernel for multi-core 32-bit systems. The Device Driver will only list those devices that are supported. It will not list any other devices then mentioned above.

Svarbu

The Device Driver can only install drivers which are available in the repositories. Make sure you have an internet connection if you want to install the drivers.

Komandos

Jeigu teikiate pirmenybę įdiegti tvarkykles per terminalą, galite naudoti šias komandas:

 

# ddm -i amd

This will install the fglrx drivers for your AMD/ATI graphical card. AMD/ATI cards from version HD 5000 and up are supported by the fglrx drivers. Versions between 1000 and 5000 will use the Radeon drivers. Use "ddm -p amd" to remove the drivers from your system.

# ddm -i nvidia

Tai, jūsų Nvidia grafikos plokštei parinks tinkamas tvarkykles. Tuo atveju, jeigu turite hibridinę plokštę (Nvidia ir Intel), tai parinks Bumblebee tvarkykles. Naudokite "ddm -p nvidia", kad pašalintumėte tvarkykles iš savo sistemos.

# ddm -i open

Jūs galite naudoti šią komandą, jeigu norite sugrįžti prie atvirų Nouveau tvarkyklių. Tai pašalins bet kokias nuosavybines tvarkykles iš jūsų sistemos.

# ddm -i broadcom

Ši komanda jūsų belaidžiam Broadcom adapteriui įdiegs tinkamas tvarkykles. Naudokite "ddm -p broadcom", kad pašalintumėte tvarkykles iš savo sistemos.

# ddm -i pae

Tai, kelių procesorių sistemose, kuriose vykdoma 32-bitų operacinė sistema, įdiegs PAE branduolį. Naudokite "ddm -p pae", kad pašalintumėte PAE branduolį iš savo sistemos. Turėkite omenyje, kad jūs negalite pašalinti branduolio, kuomet jūsų sistema yra paleista į tą branduolį.
