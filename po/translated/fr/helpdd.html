SolydXK System Settings: Aide pilote de périphérique

Description

Le Device Driver est un outil pour aider les utilisateurs à installer des drivers tiers de matériel. Actuellement, les pilotes Nvidia, AMD/ATI et Broadcom ainsi que le noyau PAE pour les systèmes 32bits multi-cœurs peuvent être installés. Le Device Driver répertorie uniquement les périphériques pris en charge. Il ne listera aucun autre périphérique que ceux mentionnés ci-dessus.

Important

Le Device Driver peut seulement installer les drivers disponibles dans les dépôts. Assurez vous d'avoir une connexion internet si vous voulez les installer.

Commandes

Si vous préferez installer des pilotes via le terminal, vous pouvez utiliser les commandes suivantes:

 

# ddm -i amd

Cela installera les drivers pour votre carte graphique AMD/ATI. Les versions HD 5000 et supérieures sont prises en charge par ces pilotes. Les versions entre 1000 et 5000 utiliseront les drivers Radeon. Utilisez "ddm -p amd" pour supprimer les pilotes de votre système.

# ddm -i nvidia

Ceci sélectionnera le bon pilote pour votre carte graphique Nvidia. Le pilote Bumblebee sera sélectionné si vous utilisez une carte hybride (à la fois Nvidia et intel). Utilisez la commande "ddm -p nvidia" pour supprimer ces pilotes de votre système.

# ddm -i open

Vous pouvez utiliser cette commande si vous souhaitez revenir à l'utilisation des pilotes libres Nouveau. Tous les pilotes propriétaires seront supprimés de votre système.

# ddm -i broadcom

Cette commande installera le bon pilote pour votre carte sans fil braodcom. Utilisez la commande "ddm -p broadcom" pour supprimer ce pilote de votre système.

# ddm -i pae

Ceci installera le noyau PAE pour les systèmes multi-processeur exécutant un système d'exploitation 32-bit. Utilisez la commande "ddm -p pae" pour supprimer ce noyau de votre système. Vous ne pourrez cependant pas supprimer ce noyau si vous l'avez utilisé pour démarrer votre système et qu'il est en cours d'utilisation.

Connexion
