SolydXK System Settings: Ajuda per als controladors

Descripció

El Device Driver és una eina per ajudar als usuaris a instal·lar controladors de maquinari de tercers. Actualment es poden instal·lar controladors nVidia, AMD/ATI i Broadcom; també es pot instal·lar el kernel PAE per als sistemes multi-nucli de 32 bit. El Device Driver només us llistarà els dispositius amb suport. No llista altres dispositius que els esmentats abans.

Important

El Device Driver només pot instal·lar controladors disponibles als dipòsits. Assegureu-vos que teniu connexió a internet si voleu instal·lar els controladors.

Ordres

Si preferiu instal·lar els controladors des del terminal, podeu usar aquestes ordres:

 

# ddm -i amd

Això instal·larà els controladors per la vostra tarja gràfica AMD/ATI. Els controladors donen suport a les targes a partir de la versió HD 5000. Les versions entre 1000 i 5000 usen els controladors Radeon. Useu l'ordre "ddm -p amd" per eliminar aquests controladors del vostre sistema.

# ddm -i nvidia

Això triarà els controladors correctes per a la vostra tarja gràfica Nvidia. També seleccionarà els controladors Bumblebee en el cas que tingueu una tarja híbrida (amb Nvidia i Intel). Useu "ddm -p nvidia" per eliminar els controladors del vostre sistema.

# ddm -i open

Podeu usar aquesta ordre per a reinstal·lar els controladors lliures Nouveau. Això eliminarà els controladors propietaris del vostre sistema.

# ddm -i broadcom

Aquesta ordre instal·larà el controlador adequat per al vostre adaptador sense fils Broadcom. Useu "ddm -p broadcom" per eliminar aquests controladors del vostre sistema.

# ddm -i pae

Això instal·larà el nucli (kernel) PAE en sistemes multiprocessador amb S. O. de 32 bit. Useu "ddm -p pae" per eliminar el kernel PAE del vostre sistema. Tingueu en compte que no es pot eliminar el nucli que s'està executant en aquell moment.

Autenticant

Fitxer: /var/log/solydxk-system.log
