SolydXK System Settings: Device Driver Help

Beschrijving

The Device Driver is a tool to help users installing third-party hardware drivers. Currently Nvidia, AMD/ATI and Broadcom drivers can be installed and allows to install the PAE kernel for multi-core 32-bit systems. The Device Driver will only list those devices that are supported. It will not list any other devices then mentioned above.

Belangrijk

The Device Driver can only install drivers which are available in the repositories. Make sure you have an internet connection if you want to install the drivers.

Commando's

Als u bij voorkeur de terminal gebruikt om de drivers te installeren, kunt u gebruik maken van de volgende commando's:

 

# ddm -i amd

This will install the fglrx drivers for your AMD/ATI graphical card. AMD/ATI cards from version HD 5000 and up are supported by the fglrx drivers. Versions between 1000 and 5000 will use the Radeon drivers. Use "ddm -p amd" to remove the drivers from your system.

# ddm -i nvidia

Dit zal de juiste drivers voor de Nvidia grafische kaart selecteren. Het zal de Bumblebee drivers selecteren in het geval u een hybride-kaart (zowel Nvidia en Intel) heeft. Gebruik "ddm -p nvidia" om de drivers van uw systeem te verwijderen.

# ddm -i open

U kunt dit commando gebruiken wanneer u terug wilt gaan naar de open Nouveau drivers. Het zal ieder driver die niet open source is van uw systeem te verwijderen.

# ddm -i broadcom

Dit commando zal de juiste driver voor uw draadloze Broadcom adapter installeren. Gebruik "ddm -p broadcom" om de drivers van uw systeem te verwijderen.

# ddm -i pae

Dit zal de PAE kernel op multi-processor systemen met een 32-bits besturingssysteem installeren. Gebruik "ddm -p pae" om de PAE kernel van uw systeem te verwijderen. Let op dat u de kernel niet kunt verwijderen wanneer u momenteel in die kernel bent opgestart.
