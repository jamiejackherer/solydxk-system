SolydXK-Systemeinstellungen: Gerätetreiber-Hilfe

Beschreibung

Der Gerätetreiber ist ein Tool, das Benutzern bei der Installation von Hardwaretreibern von Drittanbietern hilft. Aktuell können Nvidia-, AMD/ATI- und Broadcom-Treiber installiert werden und ermöglichen die Installation des PAE-Kernels für Multicore-32-Bit-Systeme. Der Gerätetreiber listet nur die Geräte auf, die unterstützt werden. Es werden keine weiteren Geräte aufgelistet, die oben erwähnt wurden.

Wichtig

Der Gerätetreiber kann nur Treiber installieren, die in den Repositories verfügbar sind. Stellen Sie sicher, dass Sie über eine Internetverbindung verfügen, wenn Sie die Treiber installieren möchten.

Befehle

Wenn Sie die Treiberinstallation im Terminal durchführen wollen, können Sie die folgenden Befehle benutzen:

 

# ddm -i amd

Dadurch werden die Treiber für Ihre AMD/ATI-Grafikkarte installiert. AMD/ATI-Karten ab Version HD 5000 werden von den Treibern unterstützt. Für Versionen zwischen 1000 und 5000 werden die Radeon-Treiber verwendet. Benutzen Sie "ddm -p amd", um die Treiber von Ihrem System zu entfernen.

# ddm -i nvidia

Dies wird die passenden Treiber für Ihre Nvidia Grafikkarte wählen. Für Hybridgrafikkarten (gleichzeitig Nvidia und Intel) werden die Bumblebee Treiber ausgewählt. Benutzen Sie "ddm -p nividia" um die Treiber von Ihrem System zu entfernen.

# ddm -i open

Dieser Befehl ermöglicht die Rückkehr zum quelloffenen Nouveau-Treiber. Er wird sämtliche proprietären Treiber von Ihrem System entfernen.

# ddm -i broadcom

Dieser Befehl wird die passenden Treiber für Ihren Broadcom W-LAN-Adapter installieren. Benutzen Sie "ddm -p broadcom" um die Treiber von Ihrem System zu entfernen.

# ddm -i pae

Dies installiert den PAE Kernel für 32-bit Betriebssysteme auf Multi-Prozessor-Systemen. Benutzen Sie "ddm -p pae" um den PAE Kernel von Ihrem System zu entfernen. Der Kernel kann nicht deinstalliert werden, während er in Benutzung ist.
